# Git: практические шаги

Этот файл описывает конкретные действия, которые можно выполнить прямо сейчас в своём проекте.

---

## 1. Создать репозиторий

### Вариант А: начать с нуля

1. Создай новую папку с проектом:  
   `mkdir my-devops-notes && cd my-devops-notes`
2. Инициализируй репозиторий:  
   `git init` [web:3].
3. Добавь файл README и конспекты в Markdown:  
   `echo "# Мои конспекты по DevOps" > README.md`

### Вариант Б: клонировать существующий репозиторий

Если репозиторий уже создан на GitHub/GitLab:

git clone git@github.com:username/devops-notes.git
